# InfoBox Official Documentation

InfoBox drives user engagement and improve the customer journey with guided walkthroughs that are seamlessly layered on your DNN portal with no code change. 

InfoBox creates digital walkthroughs on your DNN portal improving the user experience and providing personalized guidance. You can use InfoBox to guide your visitors through the website, redirect them to specific pages or encourage them to perform different actions.
